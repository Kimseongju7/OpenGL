# OpenGL 좌표계 개념

## 좌표계란?

위치를 숫자로 표현하는 기준입니다.
`glVertex2f(50, 50)`에서 50, 50이 **어디를 가리키는지** 정해주는 것이 좌표계입니다.

---

## glViewport vs gluOrtho2D

| | 역할 |
|---|---|
| `glViewport` | 창에서 그림 그릴 **픽셀 영역** 지정 |
| `gluOrtho2D` | 그 영역에 **좌표계(눈금)를 어떻게 매핑**할지 지정 |

**좌표계는 `gluOrtho2D`에 의해 결정됩니다.**

```
gluOrtho2D(0, 100, 0, 100)
     ↓ 매핑
glViewport(0, 0, 500, 500)  ← 500x500 픽셀 영역

좌표 (0,0)     = 픽셀 (0,0)
좌표 (50,50)   = 픽셀 (250,250)  ← 중앙
좌표 (100,100) = 픽셀 (500,500)
```

---

## gluOrtho2D = 눈금 긋기

`gluOrtho2D(0, 100, 0, 100)`은 창에 0~100짜리 눈금을 긋는 것과 같습니다.

```
100 ┼─────────────
    │
 50 ┼      ← 중앙
    │
  0 ┼────────────
    0    50    100
```

`gluOrtho2D(0, 200, 0, 100)`으로 바꾸면 X축 눈금이 200칸으로 늘어나서,
같은 좌표 (50, 50)이 중앙이 아니라 왼쪽으로 이동합니다.

---

## gluOrtho2D와 glRectf는 같은 좌표계 공유

`gluOrtho2D`가 눈금을 정하면, `glRectf`는 그 눈금 위의 위치를 지정합니다.

```
gluOrtho2D(0, 100, 0, 100)  ← 눈금 설정
glRectf(20, 20, 80, 80)     ← 그 눈금 위의 위치
```

```
100 ┼─────────────
    │    ┌──────┐
 80 ┼    │      │
    │    │      │
 20 ┼    └──────┘
  0 ┼────┼────┼───
    0   20   80  100
```

`gluOrtho2D`가 바뀌면 같은 `glRectf(20, 20, 80, 80)`이어도 화면에서 다른 위치/크기로 보입니다.

---

## 좌표계 범위보다 큰 도형은 잘림

```
gluOrtho2D(0, 100, 0, 100) 일 때
glVertex2f(150, 50)  ← x=150, 범위 밖 → 화면에서 잘림
```

```
(0,100)──────(100,100)
  │                 │
  │  ┌──────────────── 여기서 잘림
  │  │              │
(0,0)──────(100,0)
```

마치 카메라 밖으로 나간 것과 같습니다.

---

## 창 크기가 변하면 눈금을 다시 그림

`reshape` 함수가 창 크기가 바뀔 때마다 호출되면서 `gluOrtho2D`로 눈금을 다시 그립니다.

```cpp
void reshape(int w, int h) {
    glViewport(0, 0, w, h);          // 도화지 크기 갱신
    gluOrtho2D(0.0, 100, 0.0, 100);  // 눈금 다시 그리기
}
```

### 고정 좌표계 → 찌그러짐 발생

`gluOrtho2D(0, 100, 0, 100)`처럼 고정 범위를 쓰면,
창이 가로로 늘어나도 눈금은 여전히 0~100이라 간격이 늘어남 → **사각형 찌그러짐**

```
정사각형 창:        가로로 늘린 창:
┌──────┐           ┌────────────┐
│  □   │    →      │     □      │  ← 납작해짐
└──────┘           └────────────┘
```

### 동적 좌표계 → 찌그러짐 없음

`gluOrtho2D(0, w, 0, h)`로 하면 창 크기에 맞게 눈금 간격이 항상 유지됨 → **찌그러짐 없음**

단, 좌표 의미가 바뀜:
- `gluOrtho2D(0, 100, 0, 100)` → 좌표 50 = 항상 화면 정중앙
- `gluOrtho2D(0, w, 0, h)` → 좌표 50 = 창이 커지면 왼쪽으로 치우침
